<h2>2. 개발목적 및 프로젝트 개요</h2>

Java 개발 입문자를 위한 개발 능력 향상을 목표로 한 프로젝트 입니다.
 
 POS 프로그램을 제작했고
 
 데이터를 별도로 관리하는 DAO ,VO 클래스를 작성하였고
 
 데이터베이스를 관리하고 이벤트 핸들러를 사용했습니다.
